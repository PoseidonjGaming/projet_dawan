# Titre du projet

Application de bureau permettant la gestion de bases de données de séries.

## Description

Il existe de très nombreuses séries grâce aux développements des différentes plateformes de streaming VOD. Il est nécessaire de pouvoir centraliser toutes les informations disponibles
afin de pouvoir s'y retrouver dans toute cette offre.
